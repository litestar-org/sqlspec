Litestar + AioSQLite Application
================================

Seed a lightweight article table with SQLSpec and expose it through Litestar using the async
aiosqlite adapter.

Run locally:

.. code-block:: console

   uv run python docs/examples/frameworks/litestar/aiosqlite_app.py
